# Memory Index — Serpentos Project

> Этот каталог читается ВСЕМИ агентами: Claude Code, OpenCode, Gemini, Cursor, Zed.
> Агент должен прочитать все файлы в этой папке при старте сессии.

## Файлы

| Файл | Содержимое |
|------|------------|
| `project.md` | Контекст проекта, цели, статус |
| `stack.md` | Технический стек, порты, сервисы |
| `design.md` | Дизайн-система ectic, компоненты |
| `decisions.md` | Принятые решения (ADR) |
| `session.md` | Последняя сессия, незакрытые задачи |

## Инструкция для агента

```
Прочитай все файлы в memory/ перед началом работы.
Не переспрашивай о вещах, которые уже описаны здесь.
Обновляй memory/session.md после каждой значимой сессии.
```
